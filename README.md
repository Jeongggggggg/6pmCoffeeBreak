# 6pm Coffee Break
6pm Coffee Break는 유니티 엔진을 활용하여 만든, 대화형 게임입니다.


## 1. 소개
플레이어는 카페의 주인장으로,

손님들의 이야기와 질문에 적절한 대답을 하게 됩니다.

대화형 게임으로, 게임 오버가 되는 상황은 없으며,

매 상황마다, 플레이어는 두 가지 대답 중 하나의 대답을 선택하게 됩니다.

이러한 플레이어의 선택은 플레이어의 성격에 반영이 되며

마지막 씬에서는, 인게임 중의 선택을 종합하여 도출해 낸 플레이어의 최종적인 성격이 결과로 표시됩니다.



## 2. 메인 메뉴(Title)

![](https://github.com/Jeongggggggg/6pmCoffeeBreak/blob/main/Images/markdown/title.png?raw=true)

타이틀 화면은
* 시작하기
* 끝내기

두 버튼으로 구성됩니다.

시작하기 버튼을 누르면, 프롤로그 장면으로 넘어갑니다.

끝내기 버튼을 누르면, 화면이 닫히며 게임이 종료됩니다.

## 4. 인게임

![](https://github.com/Jeongggggggg/6pmCoffeeBreak/blob/main/Images/markdown/ingame1.png?raw=true)

플레이 화면입니다.

선택을 해야하는 상황은 총 4번입니다.

자연스러운 연출을 위해, 선택에 따라서 손님과 주인장과의 대화가 조금씩 다르도록 설정하였습니다.

## 7. 아쉬운 점 및 보완해야할 점

취지는, 대화를 통해 나타나는 본인의 성격을 알아볼 수 있는 게임을 만드는 것이었습니다. 그러나, 이러한 취지의 게임임을 알고 플레이할 경우, 선택지를 의식적으로 선택할 가능성이 있기에, 이를 해결할 수 있도록 보완해야 할 것입니다.

이를 위해, 선택 상황을 늘릴 필요가 있을 것입니다. 또, 선택지 중에서 일반적인 선택지(성격에 반영되지 않는 선택지)를 추가한다면, 플레이어가 의식적으로 선택하지 않도록 유도할 수 있을 것이라고 생각합니다.
